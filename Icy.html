Welcome to Icy Installer

Icy Installer 3 is a new, fast and lightweight package installer for jailbroken iOS devices.

 How to use Icy?

Icy is very simple to use. The text labels at the top are the name and the description of the current tab. Tabs can be switched using the buttons at the bottom. The "Home" tab is where you are right now. In the "Sources" tab you can search for packages. Just tap on a package to install it. The "Manage" tab is used to get info about existing packages, and remove them.

 Where can I get more info or support?

On my Twitter page, on the jailbreak subreddit, or the iOSThemes subreddit.
